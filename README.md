## Frontend Openapi Spec Mock Server Example

This project demonstrates how to implement a mock server based on an Openapi spec for a frontend app.

Install required dependencies:

```console
npm install
```

Generate Openapi specs:

```console
npm run gen
```

Lint specs:

```console
npm run lint
```
